# Dhiraj Integration Contract

This contract dictates exactly how output generated by Dhiraj's simulation is mapped into the global causal deterministic state stream.

## 1. Expected Extracted Schema
Dhiraj's layer performs local structural/corrosion logic. Upon interval complete, it output standard JSON formatted *exactly* by Dhiraj's contract (`MARINE-INT-002`).

```json
{
  "contract_version": "1.0.0",
  "packet_id": "550e...",
  "zones": [
    {
      "zone_id": "BOW_PORT",
      "state_transitions": {
        "delta_corrosion_mm": {
          "value": 0.00034,
          "ci_95": [0.00026, 0.00042],
          "confidence_flag": "NOMINAL"
        },
        "delta_coating_mm": {
          "value": -0.000041
        },
        "delta_roughness_um": {
          "value": 0.0052
        },
        "delta_fouling_coverage": {
          "value": 0.00012
        },
        "delta_fouling_thick_mm": {
          "value": 0.0001
        }
      }
    }
  ]
}
```

## 2. Event Translation
The Marine Adapter intercepts this JSON block. It wraps it inside an `ExecutionEvent` directly. The data structure forms the `payload`.

**Transformation Outcome:**
```python
ExecutionEvent(
    event_id="e54c-...",
    event_type="STATE_UPDATE",
    origin_node="SimulationRunner",
    payload={ "contract_version": "1.0.0", "zones": [ { "zone_id": "BOW_PORT", ... } ] },
    timestamp=162391294.0
)
```

## 3. Propagation and Application Update
Once translated:
1. The **Adapter/Client** routes this event to `POST /execute` via the Execution Interface.
2. The **Core Engine** sequences it blindly, dropping conflicts/latency issues, assigning `causal_id=N`.
3. The **Engine** proxies the payload directly back into all replicating nodes.
4. Each node routes the `payload` to their local instances of `MarineStateEngine`.
5. The **Marine Adapter** iterates through `payload["zones"]` and intercepts `state_transitions`, pulling `.get("value")` from each field and accumulating it over properties like `corrosion_depth_mm`.
6. Post-apply, a new deterministic SHA-256 hash is confirmed mirroring consensus constraint proofs.
